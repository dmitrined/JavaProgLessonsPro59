### Работа со строками (String, StringBuilder, StringBuffer)

В Java строки — это объекты, представляющие последовательность символов.

#### 1. Класс String
*   **Неизменяемость (Immutability):** Объект `String` нельзя изменить после создания. Любая операция изменения (конкатенация, замена) создает **новый** объект.
*   **String Pool (Пул строк):** Специальная область памяти в Heap для хранения строковых литералов. Позволяет экономить память, переиспользуя одинаковые строки.
    ```java
    String s1 = "Java"; // помещается в пул
    String s2 = "Java"; // ссылка на тот же объект в пуле
    String s3 = new String("Java"); // новый объект в куче (не в пуле)
    ```

**Основные методы String:**
*   `length()`: возвращает длину строки.
*   `charAt(int index)`: возвращает символ по индексу.
*   `substring(int begin, int end)`: возвращает подстроку.
*   `indexOf(String str)`: возвращает индекс первого вхождения подстроки.
*   `equals(Object obj)`: сравнивает содержимое строк (для ссылок используйте `==`).
*   `equalsIgnoreCase(String another)`: сравнение без учета регистра.
*   `toUpperCase()`, `toLowerCase()`: преобразование регистра.
*   `trim()`: удаляет пробелы в начале и конце (ASCII). `strip()` (Java 11+) — удаляет все пробельные символы Unicode.
*   `split(String regex)`: разбивает строку на массив по разделителю.
*   `replace(target, replacement)`: заменяет символы или подстроки.
*   `valueOf(type v)`: статический метод для преобразования других типов в строку.

#### 2. StringBuilder и StringBuffer
Используются для создания изменяемых строк. Эффективны при частых модификациях (например, в циклах), так как не создают множество промежуточных объектов.

*   **StringBuilder:**
    *   Не синхронизирован (не потокобезопасен).
    *   Работает быстрее.
    *   Рекомендуется для использования в однопоточной среде.

*   **StringBuffer:**
    *   Синхронизирован (потокобезопасен).
    *   Медленнее из-за накладных расходов на синхронизацию.
    *   Используется в многопоточной среде.

**Основные методы (одинаковы для обоих):**
*   `append(value)`: добавляет данные в конец.
*   `insert(offset, value)`: вставляет данные в указанную позицию.
*   `delete(start, end)`: удаляет символы.
*   `reverse()`: переворачивает строку.
*   `toString()`: возвращает неизменяемый объект `String`.

```java
StringBuilder sb = new StringBuilder("Hello");
sb.append(" World").reverse();
System.out.println(sb); // dlroW olleH
```
