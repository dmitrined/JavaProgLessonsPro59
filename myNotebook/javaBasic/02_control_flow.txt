### Управляющие конструкции: Условия и Циклы

Управляющие конструкции позволяют изменять поток выполнения программы в зависимости от условий или повторять блоки кода.

#### 1. Условные операторы

**if-else**
Используется для выполнения блока кода, если условие истинно.
```java
if (condition) {
    // код выполняется, если condition == true
} else if (anotherCondition) {
    // код выполняется, если anotherCondition == true
} else {
    // код выполняется, если ни одно условие не подошло
}
```
*   Тернарный оператор: `variable = (condition) ? expressionTrue : expressionFalse;`

**switch**
Используется для выбора одного из множества вариантов выполнения. Работает с примитивами (byte, short, char, int), их обертками, String и Enum.
*   `break` необходим в классическом `switch`, чтобы не выполнять последующие кейсы.
*   С Java 12+ доступен улучшенный `switch` (Switch Expressions), который возвращает значение и не требует `break`.

```java
// Switch Expression (Java 12+)
String dayType = switch (day) {
    case 1, 2, 3, 4, 5 -> "Weekday";
    case 6, 7 -> "Weekend";
    default -> "Invalid day";
};
```

#### 2. Циклы

**for**
Используется, когда известно точное количество итераций.
```java
for (int i = 0; i < 5; i++) {
    System.out.println("Iteration: " + i);
}
```

**while**
Выполняется до тех пор, пока условие истинно. Проверка условия происходит **перед** итерацией.
```java
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
```

**do-while**
Похож на `while`, но проверка условия происходит **после** итерации. Гарантирует выполнение тела цикла хотя бы один раз.
```java
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);
```

**for-each (Enhanced for loop)**
Удобен для перебора массивов и коллекций (реализующих `Iterable`). Не дает доступа к индексу элемента.
```java
String[] cars = {"Volvo", "BMW", "Ford"};
for (String car : cars) {
    System.out.println(car);
}
```

#### 3. Операторы перехода
*   `break`: немедленно прерывает выполнение цикла или `switch`.
*   `continue`: пропускает текущую итерацию цикла и переходит к следующей.
*   `return`: завершает выполнение метода и возвращает значение (если метод не `void`).
