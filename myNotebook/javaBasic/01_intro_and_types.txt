### Введение в Java: Основы

Java — это объектно-ориентированный, строго типизированный язык программирования общего назначения.

#### 1. JVM, JRE и JDK
*   **JVM (Java Virtual Machine)** — виртуальная машина, которая исполняет байт-код. Обеспечивает кроссплатформенность ("Write Once, Run Anywhere").
*   **JRE (Java Runtime Environment)** — среда выполнения. Содержит JVM и стандартные библиотеки классов. Нужна пользователям для запуска Java-приложений.
*   **JDK (Java Development Kit)** — полный набор инструментов разработчика. Содержит JRE + компилятор (`javac`), архиватор (`jar`), генератор документации (`javadoc`) и другие утилиты.

#### 2. Примитивные типы данных
Java имеет 8 примитивных типов данных, которые хранятся в стеке (Stack):
*   **Целые числа:**
    *   `byte` (8-bit): от -128 до 127.
    *   `short` (16-bit): от -32,768 до 32,767.
    *   `int` (32-bit): основной целочисленный тип (от -2^31 до 2^31-1).
    *   `long` (64-bit): для больших чисел (суффикс `L`, например `100L`).
*   **Числа с плавающей точкой:**
    *   `float` (32-bit): одинарная точность (суффикс `f`, например `3.14f`).
    *   `double` (64-bit): двойная точность (стандарт для дробных чисел).
*   **Символьный:**
    *   `char` (16-bit): хранит один символ Unicode (например, `'A'`, `'\u0041'`).
*   **Логический:**
    *   `boolean`: принимает значения `true` или `false`.

#### 3. Ссылочные типы данных
Это объекты (классы, интерфейсы, массивы, перечисления).
*   Переменная ссылочного типа хранит **адрес** объекта в памяти (Heap).
*   Значение по умолчанию для ссылок — `null`.
*   Примеры: `String`, `Integer` (обертка), `Scanner`, `int[]`, пользовательские классы.

#### 4. Преобразование типов (Casting)
*   **Неявное (расширение):** `byte` -> `short` -> `int` -> `long` -> `float` -> `double`. Происходит автоматически.
*   **Явное (сужение):** Требует указания типа в скобках, возможна потеря данных.
    ```java
    double d = 9.78;
    int i = (int) d; // i станет 9
    ```

#### Основные методы и операции:
```java
// Объявление и инициализация
int age = 25;
double price = 19.99;
char grade = 'A';
boolean isJavaFun = true;
String name = "Java";

// Арифметические операции
int sum = 10 + 5;
int mod = 10 % 3; // остаток от деления (1)
age++; // инкремент

// Вывод в консоль
System.out.println("Hello, Java!"); // с новой строки
System.out.print("Text ");          // без перехода
System.out.printf("Age: %d, Price: %.2f", age, price); // форматированный вывод
```
