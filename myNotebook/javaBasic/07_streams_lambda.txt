### Java Stream API и Lambda Expressions

Введено в Java 8. Позволяет писать код в функциональном стиле, упрощая обработку коллекций данных.

#### 1. Lambda Expressions (Лямбда-выражения)
Анонимные функции, которые можно передавать как аргументы или возвращать.
Синтаксис: `(параметры) -> { тело }`
*   Если параметр один, скобки можно опустить: `x -> x * 2`
*   Если тело состоит из одной строки, фигурные скобки и `return` можно опустить.

**Функциональные интерфейсы (java.util.function):**
Интерфейсы с одним абстрактным методом.
*   `Predicate<T>`: `T -> boolean` (проверка условия). Метод `test()`.
*   `Function<T, R>`: `T -> R` (преобразование). Метод `apply()`.
*   `Consumer<T>`: `T -> void` (действие над объектом). Метод `accept()`.
*   `Supplier<T>`: `() -> T` (поставщик объекта). Метод `get()`.
*   `UnaryOperator<T>`: `T -> T`.
*   `BinaryOperator<T>`: `(T, T) -> T`.

#### 2. Stream API
Поток данных, который не хранит элементы, а передает их из источника через конвейер операций.

**Основные характеристики:**
*   Не изменяет источник данных.
*   Ленивые вычисления (Lazy): промежуточные операции выполняются только при вызове терминальной.
*   Одноразовые: стрим нельзя использовать повторно после терминальной операции.

**Виды операций:**
1.  **Промежуточные (Intermediate):** Возвращают `Stream`.
    *   `filter(Predicate)`: отбирает элементы.
    *   `map(Function)`: преобразует элементы.
    *   `flatMap(Function)`: преобразует каждый элемент в стрим и объединяет их ("выпрямляет" структуру).
    *   `sorted()`: сортирует (естественный порядок или Comparator).
    *   `distinct()`: убирает дубликаты (по `equals`).
    *   `limit(n)`: берет первые n элементов.
    *   `skip(n)`: пропускает первые n элементов.
    *   `peek(Consumer)`: выполняет действие без изменения потока (для отладки).

2.  **Терминальные (Terminal):** Возвращают результат или void. Запускают поток.
    *   `collect(Collector)`: собирает в коллекцию (`Collectors.toList()`, `toSet()`, `toMap()`).
    *   `forEach(Consumer)`: применяет действие к каждому элементу.
    *   `count()`: возвращает количество элементов.
    *   `reduce(BinaryOperator)`: сводит элементы к одному значению (аккумуляция).
    *   `min()`, `max()`: поиск минимума/максимума.
    *   `findFirst()`, `findAny()`: поиск элемента (возвращает `Optional`).
    *   `anyMatch()`, `allMatch()`, `noneMatch()`: проверка условий.

#### Пример:
```java
List<String> names = List.of("Alice", "Bob", "Charlie", "Anna");

long count = names.stream()
    .filter(name -> name.startsWith("A")) // Оставить только на 'A'
    .map(String::toUpperCase)             // Преобразовать в верхний регистр
    .sorted()                             // Отсортировать
    .count();                             // Посчитать количество
```

#### Method References (Ссылки на методы)
Сокращенный синтаксис для лямбд, вызывающих существующий метод.
*   `System.out::println` вместо `x -> System.out.println(x)`
*   `String::length` вместо `s -> s.length()`
