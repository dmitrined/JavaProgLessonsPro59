Инструкция: Как сделать Push в GitHub

1. Проверить статус изменений  Команда: git status

2. Добавить изменения в индекс (Staging Stage) Команда: git add .

3. Создать коммит (Зафиксировать изменения) Команда: git commit -m "Ваше описание изменений"

4. Отправить изменения в удаленный репозиторий (GitHub) Команда: git push

Дополнительные команды:
- git pull — затянуть свежие изменения из GitHub в локальный проект (делать перед началом работы).
- git remote -v — проверить, к какому репозиторию привязан проект.
