1.9 Типовые ошибки при работе с Entity

- Использовать Entity как DTO: допустимо в обучении, плохо в продакшене.
- equals / hashCode по id: id появляется только после сохранения, опасно для коллекций.
- LazyInitializationException: доступ к lazy-связям вне транзакции.
- Циклические JSON-ссылки: User → Orders → User...
  Решения: DTO, @JsonIgnore, правильная архитектура.
