clean удаляет результат предыдущей сборки типично clean package
validate быстрая проверка что проект валидный, есть валидный pom и базовая структура
compile компиляция src/main/java --> targen/classes быстрая проверка компиляции кода
test запускает unit-тесты типично перед package
package собирает артефакт проекта и кладет его в target
verify проверка качества сборки, валидация артефактов, плагины контроля
install кладет артефвкт проекта в локальный репозиторий
site документация и отчеты
deploy публикует артефакт на удаленном репозитории, делает релизную версию. Альтернатива: Jenkins, Tekton
mvn dependency:tree вывод на экран дерева зависимостей
